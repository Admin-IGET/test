How to Download UvikOS

Welcome to the UvikOS download guide.
This guide will walk you through downloading UvikOS. It also includes images.

First, open the UvikOS download link (preferably in a different window): UvikOS - Google Drive. Then right-click the UvikOS_en folder and then select the Download option.

Wait for the ZIP archive to download (for example, "UvikOS_en-20260607T091612Z-3-001.zip"). After the download is complete, locate the file in File Explorer. You can do this by clicking the folder icon next to the downloaded file.

Extract the downloaded ZIP archive. You can do this by right-clicking the file, selecting Extract All, and then clicking Extract.

Next, locate the extracted folder and run the "Install UvikOS" file. Because UvikOS doesn't have a paid certificate, Windows may show a SmartScreen warning. Just click More info and then Run anyway to run the file.
