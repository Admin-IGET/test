Nabídka Změnit pozadí plochy
Jméno souboru: DBackground.exe Typ: Viditelný program Programovací jazyk: C# Datum vytvoření: 13.12.2025
Tento program změní pozadí plochy na obrázek, který vyberete. Má dva režimy. Režim normálního spuštění: Po výběru obrázku se UvíkOS restartuje a pozadí se nastaví. Režim spuštění přes program Personalizace a nastavení barev: Po výběru obrázku se nabídka zavře a můžete pokračovat v měnění ostatních nastavení.
