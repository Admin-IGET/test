Smart app control and UvikOS

UvikOS can crash (or the "Application control policy has blocked this file" error) if you have smart app control on.

Why?
 UvikOS doesn't have a digital certificate, because certificates aren't free. That's why smart app control can't verify the developer. That can crash the UvikPanel or show the error message above.
How to fix it?
 We recommend using this procedure:
1. Turn off smart app control
2. Open the blocked app
3. Turn smart app control back on
 Then the blocked program should open even if smart app control is enabled. If this procedure doesn't work, try restarting your PC or add the blocked file to the exceptions list, alternatively turn off smart app control.
