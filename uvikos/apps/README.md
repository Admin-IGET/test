# UvíkOS - Custom Desktop Environment

UvíkOS is a custom desktop environment for Windows that provides an alternative interface to the standard Windows desktop. It's designed to be a lightweight, customizable replacement for the Windows desktop experience.

## What This Project Does

- **Custom Desktop Interface**: Provides a personalized desktop environment with custom wallpaper management
- **Icon Management**: Allows drag-and-drop icon placement with grid snapping
- **Custom Applications**: Includes built-in applications like calculator, text editor, and task manager
- **System Integration**: Seamlessly integrates with Windows while providing a custom experience

## Features

- Custom wallpaper management with multiple display modes
- Drag-and-drop desktop icons with grid alignment
- Built-in applications (Calculator, Text Editor, Task Manager, etc.)
- Customizable color schemes and transparency
- Custom taskbar and panel functionality
- WiFi management tools
- Screen saver functionality

## Installation

1. Download the project files
2. Run `install.cmd` as administrator
3. Follow the installation prompts

## System Requirements

- Windows 7 or later
- Java Runtime Environment (included in installation)
- Administrator privileges (required for system customization)

## Security Notice

This software modifies system settings to provide a custom desktop environment. All source code is available for inspection and contains no malicious functionality. The software:

- Creates directories in `C:\apps` for application files
- Modifies desktop and wallpaper settings
- Uses standard Windows APIs for system integration
- Does not collect or transmit any data

## File Structure

- **Java Applications**: Custom-built applications in Java
- **PowerShell Scripts**: Desktop environment management
- **Batch Files**: Installation and system setup
- **Executables**: Compiled applications for specific functionality

## Development

This project is developed in Java and PowerShell, with some C# components for advanced functionality. All source code is available for review and modification.
