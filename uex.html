UEX Spouštěč je program pro otevíraní .UEX ( Uvik Executable ) souborů, což jsou spustitelné soubory pro UvíkOS.

UEX Loader is a program for openning .UEX ( Uvik Executable ) files, which are Executable files for UvikOS.

Pokud se program UEX Loader nespouští, Zkuste se vrátit do windows a otevřít ho znovu. Potom by měl fungovat správně i v UvíkOS.

If the UEX Loader program isn't staring, Try to go back to windows and open it again. Afterwards, it shloud work normally even in UvikOS.
