UvikOS Download Page

You can download UvikOS below.

UvikOS works on Windows 7, 8, 8.1, 10 and 11.

Note: your Windows taskbar must not be set to auto-hide and must be positioned at the bottom of the screen.

If you're updating UvikOS, return to Windows before doing so.
Because UvikOS isn't digitally signed, SmartScreen warnings may appear. To bypass them, click More info, then Run anyway.
If Microsoft Edge shows a warning, click the … button next to the file, then Keep, then the ▼ icon, then choose Keep anyway.
I cannot guarantee that UvikOS will work properly if you are using an antivirus such as AVG or Avast. More info
If Windows Defender says the file is malicious, this may be a false positive. You may need to temporarily disable real-time protection.
If a program does not open, try running it in Windows first and then again in UvikOS.
UvikOS is not fully compatible with touchpad gestures.
