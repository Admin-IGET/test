UvikOS works on Windows 7, 8, 8.1, 10 and 11.

Note: your Windows taskbar must be positioned at the bottom of the screen.

If you're updating UvikOS, return to Windows before doing so.
Some parts of UvikOS might not work properly if you have Smart App Control enabled. How to fix?
UvikOS can be slow or partially dysfunctional if you are using an antivirus such as AVG or Avast. More info
If a program does not open, try running it in Windows first and then again in UvikOS.
It is recommended not to touch the computer while switching into or out of UvikOS.
UvikOS is not fully compatible with touchpad gestures.
UvikOS can look weird if your DPI scale is different from 100 %.

Warning

Some antivirus programs may incorrectly mark the UvikOS installation archive (ZIP) as suspicious. UvikOS contains multiple executable programs, scripts, and supporting files, which can sometimes trigger heuristic antivirus detections even when no malicious software is present.
Services such as VirusTotal may also show results for ZIP archives that reflect the archive as a whole rather than each individual file inside it, which can sometimes trigger heuristic antivirus detections.

If your antivirus displays a warning, you may review the file and decide for yourself whether you wish to continue with the download.

If you decide not to download UvikOS, you can try an alternative called Cairo Shell. However, it is more focused on technical use cases and is less customizable than UvikOS.

You need PowerShell 5.1 and .NET 4.6.

If your computer doesn't have those installed, download the update pack for Windows here. (Windows 10 and 11 already include it.)

UvikOS is completely ad-free and free of charge.

UvikOS is made in C# using PowerShell and Visual Studio 2019. Some parts use Command Prompt scripts (.cmd). All code can be edited using Notepad++ or dnSpy.

Installing UvikOS doesn't require administrator privileges.

UvíkOS doesn’t permanently modify your Windows installation; instead, it uses its own UI elements for better customization.
UvikOS isn’t digitally signed, so SmartScreen warnings may appear.
